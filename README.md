# Order-service

Development server Navigate to http://localhost:8080/.

Technology Stack :
-----------------

* Java 8
* Apache Tomcat/9.0.37
* H2 db (H2 console available at '/h2-console'. Database available at 'jdbc:h2:mem:order_service')
    * user name : sa
    * password : ""
* Spring Boot 2.3.3.RELEASE
* Apache Maven
